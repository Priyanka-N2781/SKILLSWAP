## 📌 Project Overview

**SKILLSWAP** is a full-stack web application developed to connect individuals who are willing to **teach a skill** and those who are eager to **learn a new skill**.

The platform creates a collaborative environment where users can share their knowledge, discover experts, and build a strong peer-to-peer learning community.

Instead of spending money on expensive courses, users can directly interact with real people and exchange practical knowledge.

This project was built with the vision of:

> **"Learn Together • Teach Together • Grow Together"**

---

## 🎯 Problem Statement

In today's digital era, many learners struggle to find affordable mentors while many talented individuals do not have a proper platform to showcase and share their skills.

SKILLSWAP solves this issue by providing:

- A centralized skill sharing marketplace
- Easy user-to-user interaction
- Learning through community collaboration
- Opportunity for both teaching and learning

## 🏗️ System Workflow

1. User creates an account on the platform  
2. Logs in securely  
3. Adds the skills they can offer  
4. Searches for skills they wish to learn  
5. Sends collaboration/skill exchange request  
6. Connects with the matched user  
7. Learns, teaches, and grows through interaction

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/Priyanka-N2781/SKILLSWAP-Project.git
cd SKILLSWAP-Project
```

### Install Frontend Dependencies

```bash
cd client
npm install
npm start
```

### Install Backend Dependencies

```bash
cd server
npm install
npm run dev
```
