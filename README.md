# SkillSwap

A full-featured skill exchange Android application with backend.

## Tech Stack
- **Backend**: Node.js, Express, MongoDB
- **Frontend**: Android (Java)
- **Database**: MongoDB Atlas
- **Hosting**: Netlify

## Setup

### Backend
1. `cd backend`
2. `npm install`
3. Create `.env` file with MONGO_URI and JWT_SECRET
4. `npm start`

### Android
1. Open in Android Studio
2. Build APK
